    Сейчас мы активно работаем над рядом проектов:

    Облачная инфраструктура для накопления и анализа промышленных интеллектуальных систем "Лоджик Илд"

    • Создание облачной инфраструктуры на основе базы данных (PostgreSQL), API (FastAPI) и интерфейса пользователя (RestAPI).
    • Бесшовное подключение всех устройств Лоджик Илд к облачной инфраструктуре по защищенному каналу за корпоративным VPN.
    • Возможность предварительной обработки и анализа данных (построение временных рядов, интеграция в Jupyter).

    API инфраструктура для любых создаваемых нейронных сетей:

    • Интеграция GET, POST и любых других запросов по требованиям пользователей.
    • Мониторинг API (Grafana).
    • Защищенный доступ.

    Устройство "UVYield" для анализа сыпучих материалов под ультрафиолетовым излучением

    • Анализ силы и равномерности флуоресценции гранул при УФ облучении.
    • Анализ размера, формы, цвета гранул.
    • Возможность установки устройства в производственные лаборатории.
