    DotPulse

    DotPulse

    Программа разработана для приборов «DotPulse». В основе расчетов лежит оптическое изображение слоя гранул и алгоритм сегментации каждой гранулы на изображении и аппроксимации ее эллипсом.

    ПО способно:

    • проводить расчет физических свойств гранул на изображении (линейные размеры, цвет, форма);
    • получать и отображать изображения с USB-камеры прибора «DotPulse» (включая высокоскоростные промышленные камеры);
    • проводить предварительную обработку изображений и использовать классические и нейросетевые алгоритмы для сегментации каждой гранулы;
    • создавать SQL базу данных изображений и рассчитанных свойств с возможностью импорта в *.csv;
    • выполнять анализ статических и динамических изображений гранул (в т.ч. c приборов, установленных над конвейерной лентой, на течке или пересыпке гранул продукта).

    GOA API

    GOA API

    API для предоставления данных и использования нейросетевых алгоритмов для анализа изображений гранул удобрений.

    • 3447 изображений гранул 4 типов удобрений (ДАФ, NPK 15-15-15, NH4NO3, KCl);
    • возможность отделения гранул от фона с помощью нейросетевой модели;
    • определение каждой отдельной гранулы с ее свойствами (инстанс-сегментация) с помощью нейронных сетей;
    • возможность анализа гранул верхнего слоя гранул вне зависимости от нижних слоев и фона;

    Нейросетевой анализ номерных знаков автомобилей

    Нейросетевой анализ номерных знаков автомобилей

    • Определение положения номерного знака для вида от 1-го и от 3-го лица (камеры мобильных телефонов, регистраторов и камер наблюдения).
    • Проведение OCR анализа номерного знака для получения текста и региона/страны выпуска.
    • Возможность реализации API сервиса, в том числе для локальной установки.

    Нейросетевой анализ и мониторинг спутниковых снимков

    Нейросетевой анализ и мониторинг спутниковых снимков

    • получение спутниковых данных из открытых источников (Google EE, Copernicus).
    • Проведение классификации изображений.
    • Построение временных рядов из наблюдений.
    • Поиск и построение корреляций спутниковых снимков и результатов анализа наземных данных (дорожная сцена, лаборатории по РМ2.5, РМ10, промышленное производство).
    • Возможность реализации API сервиса, в том числе для локальной установки.
